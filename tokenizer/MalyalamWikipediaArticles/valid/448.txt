പഞ്ചാബ് സംസ്ഥാനത്തെ ലുധിയാന ജില്ലയിലെ ഒരു വില്ലേജാണ് ചക് ഡാങ്കൊ. ചക് ഡാങ്കൊ വില്ലേജിന്റെ പരമാധികാരി സർപഞ്ചാണ്. ജനങ്ങൾ തെരഞ്ഞെടുക്കുന്ന പ്രതിനിധിയാണ് സർപഞ്ച്.
2011 ലെ ഇന്ത്യൻ കാനേഷുമാരി വിവരമനുസരിച്ച് ചക് ഡാങ്കൊ ൽ 0 വീടുകൾ ഉണ്ട്. ആകെ ജനസംഖ്യ 0 ആണ്. ഇതിൽ 0 പുരുഷന്മാരും 0 സ്ത്രീകളും ഉൾപ്പെടുന്നു. ചക് ഡാങ്കൊ ലെ സാക്ഷരതാ നിരക്ക് 0 ശതമാനമാണ്. ഇത് സംസ്ഥാന ശരാശരിയായ 75.84 ലും താഴെയാണ്. ചക് ഡാങ്കൊ ലെ 6 വയസ്സിനു താഴെയുള്ള കുട്ടികളുടെ എണ്ണം 0 ആണ്. ഇത് ചക് ഡാങ്കൊ ലെ ആകെ ജനസംഖ്യയുടെ 0 ശതമാനമാണ്. 
2011 ലെ ജനസംഖ്യാ കണക്കെടുപ്പ് രേഖകൾ പ്രകാരം 0 ആളുകൾ വിവിധ തൊഴിലുകളിൽ ഏർപ്പെട്ടിരിക്കുന്നു. ഇതിൽ 0 പുരുഷന്മാരും 0 സ്ത്രീകളും ഉണ്ട്. 2011 ലെ കാനേഷുമാരി പ്രകാരം 0 ശതമാനം ആളുകൾ അവരുടെ ജോലി പ്രധാന വരുമാനമാർഗ്ഗമായി കണക്കാക്കുന്നു എന്നാൽ 0 ശതമാനം പേർ അവരുടെ ഇപ്പോഴത്തെ ജോലി അടുത്ത 6 മാസത്തേക്കുള്ള താത്കാലിക വരുമാനമായി കാണുന്നു.
