പഞ്ചാബ് സംസ്ഥാനത്തെ ഷഹീബ്സദ അജിത് സിംഗ് നഗർ ജില്ലയിലെ ഒരു വില്ലേജാണ് ഭഗ്‌വാൻപൂർ. ഷഹീബ്സദ അജിത് സിംഗ് നഗർ ജില്ല ആസ്ഥാനത്തുനിന്നും 10 കിലോമീറ്റർ അകലെയാണ് ഭഗ്‌വാൻപൂർ സ്ഥിതിചെയ്യുന്നത്. ഭഗ്‌വാൻപൂർ വില്ലേജിന്റെ പരമാധികാരി സർപഞ്ചാണ്. ജനങ്ങൾ തെരഞ്ഞെടുക്കുന്ന പ്രതിനിധിയാണ് സർപഞ്ച്.
2011 ലെ ഇന്ത്യൻ കാനേഷുമാരി വിവരമനുസരിച്ച് ഭഗ്‌വാൻപൂർ ൽ 0 വീടുകൾ ഉണ്ട്. ആകെ ജനസംഖ്യ 0 ആണ്. ഇതിൽ 0 പുരുഷന്മാരും 0 സ്ത്രീകളും ഉൾപ്പെടുന്നു. ഭഗ്‌വാൻപൂർ ലെ സാക്ഷരതാ നിരക്ക് 0 ശതമാനമാണ്. ഇത് സംസ്ഥാന ശരാശരിയായ 75.84 ലും താഴെയാണ്. ഭഗ്‌വാൻപൂർ ലെ 6 വയസ്സിനു താഴെയുള്ള കുട്ടികളുടെ എണ്ണം 0 ആണ്. ഇത് ഭഗ്‌വാൻപൂർ ലെ ആകെ ജനസംഖ്യയുടെ 0 ശതമാനമാണ്. 
2011 ലെ ജനസംഖ്യാ കണക്കെടുപ്പ് രേഖകൾ പ്രകാരം 0 ആളുകൾ വിവിധ തൊഴിലുകളിൽ ഏർപ്പെട്ടിരിക്കുന്നു. ഇതിൽ 0 പുരുഷന്മാരും 0 സ്ത്രീകളും ഉണ്ട്. 2011 ലെ കാനേഷുമാരി പ്രകാരം 0 ശതമാനം ആളുകൾ അവരുടെ ജോലി പ്രധാന വരുമാനമാർഗ്ഗമായി കണക്കാക്കുന്നു എന്നാൽ 0 ശതമാനം പേർ അവരുടെ ഇപ്പോഴത്തെ ജോലി അടുത്ത 6 മാസത്തേക്കുള്ള താത്കാലിക വരുമാനമായി കാണുന്നു.
ഭഗ്‌വാൻപൂർ ലെ 0 പേരും പട്ടിക ജാതി വിഭാഗത്തിൽ പെടുന്നു. 0 പേർ പട്ടിക വർഗ്ഗ വിഭാഗത്തിലുള്ളവരാണ്.
